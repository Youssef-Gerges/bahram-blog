# Bahram Blog

simple blog with reactjs and typeScript

## Libraries used

- reactjs
- react router dom
- react icons
- JSON-SERVER

## Pages

- Home
- New Blogs
- Popular Blogs
- Reading Lists

## Running App

- Clone repository, `then execute`:

```
$ cd bahram-blog
$ npm install
$ npm install -g json-server
$ json-server --watch src\DB\db.json
$ npm run dev
```
